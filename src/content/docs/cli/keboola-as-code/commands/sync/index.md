**Synchronization between a [local directory](/cli/keboola-as-code/structure/) and a [project](/cli/keboola-as-code/#subsystems).**

```
kbc sync [command]
```

|---
| Command | Description
|-|-|-
| [kbc sync init](/cli/keboola-as-code/commands/sync/init/) | Initialize a new local directory and run `kbc sync pull`. |
| [kbc sync pull](/cli/keboola-as-code/commands/sync/pull/) | Sync a project to the local directory. |
| [kbc sync push](/cli/keboola-as-code/commands/sync/push/) | Sync the local directory to the project. |
| [kbc sync diff](/cli/keboola-as-code/commands/sync/diff/) | Show differences between the local directory and the project. |
