[R](https://cran.r-project.org/) transformations are for **advanced statistical computations** — ready-made algorithms and vector/matrix math — complementing Python and SQL where those are too difficult. Common operations like joining, sorting, or grouping are still easier and faster in [SQL transformations](/transformations/#backends). Keboola maps your input tables in as CSV, runs your script, and writes CSV outputs back to [Storage](/storage/tables/).

This page is split by what you need:

- **[How do I run an R transformation?](/transformations/r-plain/how-to/)** — create, run, and develop/debug one end to end.
- **[R transformation reference](/transformations/r-plain/reference/)** — runtime version and limits, packages, CSV format, warnings-as-errors, and logging.

In-depth examples: [array splitting](/transformations/r-plain/array-splitter/) · [charts & graphs](/transformations/r-plain/plots/) · [binary files](/transformations/r-plain/binary/).

New to transformations? Start with [Transformations](/transformations/) and the [Getting Started tutorial](/tutorial/manipulate/).
